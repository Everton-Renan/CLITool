1. Criar projetos padrao
	Cria pastas e arquivos ja preenchidos
	Linguagens: Python, PHP, C.
	Python:
		if __name__ == "__main__":
			print("Hello World")

	PHP:
		<?php
			echo "Hello World";
		?>

	C:
		#include <stdio.h>

		int main(void)
		{
			printf("Hello World\n");
		}

2. Busca e abertura de projetos
	envia o nome do projeto e ele e aberto no VSCode

3. Cria um ambiente virtual python ou sobe um servidor php
	Python: cria um venv e instala dependencias caso haja um arquivo requirements.txt
	PHP: php -S localhost:8000

4. DocSearch - Buscador automatico de documentação oficial
	Recebe uma linguagem e um termo e abre a documentação oficial no navegador

5. GitStatus: Mostra as estatísticas do GitHub
	Puxa seu número de repositórios, seguidores e nome.
